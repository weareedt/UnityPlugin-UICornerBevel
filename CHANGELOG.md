# Changelog

All notable changes to this package are documented here. This project adheres to
[Semantic Versioning](https://semver.org/).

## [1.0.0] - 2026-06-16
### Added
- Initial standalone release of **UI Corner Bevel**.
- `UICornerBevel` component (namespace `EDT.UI`) — SDF corner rounding for `Image` /
  `RawImage`, with `Corner Radius` and `Edge Softness` inspector fields.
- `UI/CornerBevel` shader.
- Optional, dependency-free `RequestRebuild` notification to a companion shadow/glow
  component via `SendMessage`.
